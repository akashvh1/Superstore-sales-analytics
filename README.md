# Superstore Sales Analytics — Task 1
*Data Analytics & Business Intelligence Internship | Maincrafts Technology*

## Overview
End-to-end data analytics project on a Superstore Sales dataset covering
data cleaning, exploratory data analysis, SQL querying, Power BI dashboard
and a business insights report.

## Files in this Repository
- raw_sales_data.xlsx — Original messy dataset
- cleaned_sales_data.xlsx — Cleaned data with EDA pivot summaries
- sales_data.csv — CSV export for SQL import
- superstore_queries.sql — SQL schema 
- Business_Insights_Report.docx 
- dashboard_screenshot.png — Final Power BI dashboard

## Tools Used
Excel | MySQL | Power BI |

## Key Insights
- North leads total sales ($69,914) but East has the highest profit margin (14%)
- Furniture drives 61% of revenue; Technology has the best margin at 14%
- Orders at 30% discount are loss-making — avg profit of −$4.51 per order
- April is the peak sales month; January and June are the weakest
